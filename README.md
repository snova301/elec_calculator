# 概要

このアプリは電気設備計算に役に立つアプリです。
サービスは予告なく停止することがあります。

【主な機能】
- 電気容量、電圧、力率、ケーブル長さを入力
- 電流値、ケーブルのサイズ、電圧降下、ケーブルの電力損失を自動計算
- 計算方法を記載

使用上の注意や使用方法については[こちら](https://snova301.github.io/AppService/)を参照ください。
